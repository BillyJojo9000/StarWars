# Немного о Звездных Воинах

Во вселеной есть множество ключевых персонажей такие как:

1. Магистр Йода
2. Энекин Скайвокер
3. Обиван Кеноби
4. Чубака

Для противостояния они используют:

1. Световые мечи
2. Лазерные бластеры
3. Силу телекинеза
4. Удачу
5. Сообразительность
